# UI Preferences

- Keep the UI minimal, calm, and modern.
- Avoid boxy or blocky layouts.
- Avoid heavy shadows, glows, glossy effects, and loud gradients.
- Prefer spacing, alignment, typography, and contrast over decoration.
- Prefer responsive layouts that stay on one row when there is space and wrap naturally when needed.
- Use clean surface separation with subtle borders and background contrast.
- Avoid too many divider lines; prefer spacing before adding separators.
- Avoid too many nested cards or boxed sections.
- Keep labels quiet and values clear.
- Use accent color sparingly.
- Keep motion subtle and purposeful.
- When in doubt, remove a container or visual effect.
